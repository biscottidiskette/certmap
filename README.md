# CertMap

CertMap is a web application for building and evaluating cybersecurity certification roadmaps.

Users can add certifications, mark them as owned or planned, and receive an AI-generated grade of their roadmap based on cost, difficulty, market relevance, and coherence.

---

## Stack

- Frontend: React + Vite
- Backend: Node.js + Express
- Storage: LocalStorage (client-side)
- AI: Groq (roadmap analysis)

---

## Features

- Build certification roadmaps
- Mark certifications as owned or planned
- Estimate cost and timeline
- AI-generated roadmap grading
- Role-based certification suggestions

---
